# 📊 Superstore Sales & Profit Analysis Dashboard

## 🚀 Overview

This project focuses on analyzing retail sales data to understand **sales performance, profitability, and customer trends**.
The objective is to identify key business insights and support data-driven decision-making using **SQL and Power BI**.

---

## 🎯 Objectives

* Analyze sales and profit trends across different regions and categories
* Identify loss-making products and the impact of discounts on profitability
* Build interactive dashboards for business monitoring
* Derive actionable insights to improve revenue and margins

---

## 📂 Dataset

* Dataset: Superstore Sales Data
* Records: **10,000+ transactions**
* Features include:

  * Sales, Profit, Discount
  * Category, Sub-category
  * Region, Segment
  * Order and Delivery details

---

## 🛠️ Tools & Technologies

* **SQL** → Data extraction and analysis
* **Power BI** → Dashboard creation and visualization
* **Python (optional)** → Data preprocessing

---

## 📊 Key KPIs

* Total Sales
* Total Profit
* Profit Margin
* Order Count
* Average Order Value (AOV)

---

## 📈 Key Insights

* Identified **loss-making sub-categories** due to high discount levels
* Observed that **discounts above ~40% significantly reduce profitability**
* Certain regions showed **higher sales but lower profit margins**
* A small number of categories contributed to the majority of revenue

---

## 📊 Dashboard Features

* Interactive filters for:

  * Region
  * Category
  * Segment
* Drill-down capability for detailed analysis
* KPI cards for quick business overview
* Visualizations:

  * Sales vs Profit trends
  * Category-wise performance
  * Regional analysis

---

1. Data cleaning and preprocessing

4. Dashboard design in Power BI
